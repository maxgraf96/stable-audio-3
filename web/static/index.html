A note on compatibility. Vocal separation runs locally in your browser via Demucs (ONNX Runtime Web). It works best on desktop Chrome or Edge, where WebGPU is available; it falls back to a slower WASM path on Safari and Firefox, and may not run reliably on iOS. If separation fails on your setup, turn off both vocal toggles to remix the full track directly.
